# ゲームデザインドキュメント

## デザイン目標

### 1ランの所要時間

| 項目 | 目標値 |
|------|--------|
| 平均所要時間 | 10〜15分 |
| 通常階層 | 30〜45秒/フロア |
| ボス階層 | 1〜2分/フロア |

### 理想的な到達階層分布

| プレイヤー層 | 到達階層 |
|-------------|---------|
| 初心者 | 3〜5F |
| 中級者 | 5〜8F |
| 熟練者 | 10Fクリア |

### デザイン判断の軸

- **戦略性優先**: 手軽さよりも1手の重みを重視
- **ランダム性と再現性のバランス**: ダンジョン構造はランダムだが、敵行動パターンは予測可能
- **視認性**: ドット絵でも敵種別・地形が一目で区別できる
- **リスクリワード**: 深く潜るほど報酬は大きいが、失うものも大きい。帰還判断に常に葛藤がある
- **周回の達成感**: 拠点アップグレードにより、周回ごとに着実な成長を実感できる
