# ビジョン / ゲーム概要

## ジャンル

ドット絵タイル制ローグライクダンジョン攻略ゲーム

## コアコンセプト

- ランダム生成ダンジョンの「探索と発見」
- ターン制の「位置取り・間合い管理」による戦略的な戦闘
- HP・アイテムの「リソース管理」と撤退判断
- 拠点の「成長・アップグレード」による周回ごとの達成感
- シンプルなドット絵で表現される、視認性の高いゲーム画面

## ゲームループ

```
拠点（アップグレード・準備）
  ↓
ダンジョン探索（戦闘・アイテム収集）
  ↓
帰還 or ゲームオーバー
  ↓
拠点（アイテム換金・アップグレード）
  ↓
再挑戦...
```

## プレイヤー体験

- 敵との間合いを考えながらフロアを探索し、戦闘の位置取りが楽しさの中心
- HP・アイテムの残量と相談しながら「もう1フロア潜るか、撤退するか」を判断
- 未知のマップ探索・罠の発見・アイテムの取捨選択がリソース管理の一部として機能する
- 階層が深くなるにつれ、敵の種類と配置が変化し緊張感が増す
- ダンジョンで得たアイテムを拠点に持ち帰り、換金して拠点をアップグレードする

## 帰還と敗北

- **帰還**: 前の階層に戻る階段に再び到達すると帰還。所持アイテムをすべて持ち帰れる
- **ゲームオーバー**: HPが0以下。最も価値の高いアイテム1つだけを握りしめて拠点に帰還

## 拠点

- ダンジョンで得たアイテムをゴールドに換金
- ゴールドを使って拠点施設をアップグレード（詳細は別途策定）
