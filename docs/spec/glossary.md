# 用語集

| 用語 | 定義 |
|------|------|
| ターン | プレイヤーの1行動 + 全敵の1行動で1ターン |
| フロア | ダンジョンの1階層。各フロアは独立したマップを持つ |
| 階層遷移 | 下り階段を通じて次のフロアに進むこと |
| タイル | マップの基盤となる最小単位（床）。8x8ピクセル |
| オブジェクト | タイルの上に配置されるもの（壁・階段・罠・宝箱など） |
| 壁 | 通行不可のオブジェクト |
| 下り階段 | 次のフロアへの遷移ポイントとなるオブジェクト |
| 上り階段 | 帰還ポイントとなるオブジェクト |
| 罠 | 踏むとダメージを受けるオブジェクト |
| 宝箱 | アイテムを入手できるオブジェクト |
| 拠点 | ダンジョン外の安全地帯。アップグレード施設がある |
| 帰還 | 上り階段から拠点に戻ること。所持アイテムをすべて持ち帰れる |
| ゴールド | アイテム換金で得られる通貨。拠点のアップグレードに使用 |
| HP | ヒットポイント。0以下でゲームオーバー |
| ゲームオーバー | HPが0以下になること。最も価値の高いアイテム1つだけ持ち帰れる |
| ラン | 拠点出発からゲームオーバーまたは帰還までの1プレイ |
| 隣接 | 4近傍（上下左右）で接しているマスの関係 |
| 索敵範囲 | 敵がプレイヤーを検知できる範囲。敵タイプにより条件が異なる |
| マンハッタン距離 | \|x1-x2\| + \|y1-y2\| で計算される距離 |
| BFS | 幅優先探索。敵の経路探索に使用 |
| 発話 | キャラクターが場面に応じてセリフを発すること |
| 頭上吹き出し | ダンジョン内でキャラクターの頭上に表示される短いセリフ |
| HUD下部テキスト | 画面下部に表示されるやや長めのセリフ。拠点・メタ的発話で使用 |
| キャラクター性格 | プレイヤーキャラクターに割り当てられる性格。発話の口調に影響する |
