# マップ生成仕様

## アルゴリズム概要

ランダム部屋配置 + 通路接続方式。

## 生成手順

1. マップ全体を壁で初期化
2. 最大 `DUNGEON_MAX_ROOMS` 個の部屋をランダムに配置
3. 各部屋のサイズは `DUNGEON_MIN_ROOM_SIZE` 〜 `DUNGEON_MAX_ROOM_SIZE` の範囲
4. 部屋同士が重ならないように配置（重なった場合はスキップ）
5. 前の部屋と新しい部屋の中心を十字形通路で接続
6. 最初の部屋に上り階段、最後の部屋に下り階段を配置

## マップ構造

マップはタイル層とオブジェクト層の2層で構成される。

### タイル層

すべてのマスは床タイルで構成される。マップの基盤。

### オブジェクト層

タイルの上に配置されるもの。1マスに最大1つのオブジェクト。

| オブジェクト | 通行 | 説明 |
|-------------|------|------|
| 壁 (WALL) | 不可 | 部屋・通路を区切る構造物 |
| 下り階段 (STAIRS_DOWN) | 可 | 次の階層への遷移ポイント |
| 上り階段 (STAIRS_UP) | 可 | 前の階層への帰還ポイント |
| 罠 (TRAP) | 可 | 踏むとダメージ（詳細は今後策定） |
| 宝箱 (CHEST) | 可 | アイテムを入手できる（詳細は今後策定） |

## 通路接続

2つの部屋の中心座標を結ぶ十字形（L字型）の通路を生成する。
水平方向を先に掘り、次に垂直方向を掘る（またはその逆をランダムに選択）。

## 敵配置

- 各部屋にランダムに敵を配置（最初の部屋を除く）
- 配置数と種類は階層に応じて変化（`enemies.md` 参照）
