# ゲームルール

## ターン進行

1. **プレイヤーターン**: 移動・攻撃・待機のいずれかを選択
2. **敵ターン**: 全ての敵が行動（移動 or 攻撃）
3. 1に戻る

プレイヤーが行動するまで敵は動かない（入力待ち型ターン制）。

## 移動

- 4方向（上下左右）に1マスずつ移動
- オブジェクト（壁など）があるマスには移動不可
- 敵がいるマスにも移動不可

## 戦闘

### プレイヤーの攻撃

- 隣接する敵に対して攻撃可能
- ダメージ = プレイヤーの攻撃力

### 敵の攻撃

- 隣接するプレイヤーに対して攻撃
- ダメージ = 敵の攻撃力

### 戦闘の解決

- プレイヤーが攻撃 → 敵が生存していれば反撃
- 1ターン1回の攻撃

## マップ

マップはタイル層とオブジェクト層の2層で構成される。

- **タイル層**: すべてのマスは床タイルで構成される
- **オブジェクト層**: タイルの上に配置されるもの（壁・階段・罠・宝箱など）

詳細は `mapgen.md` を参照。

## 階層遷移

- **下り階段**: 次の階層に進む
- **上り階段**: 前の階層に戻る階段に再び到達すると帰還判定（ダンジョンを脱出し拠点に戻る）

## 帰還・敗北

- **帰還**: 上り階段に到達して拠点に戻る。所持アイテムをすべて持ち帰れる
- **ゲームオーバー**: HP が 0 以下になる。最も価値の高いアイテム1つだけを持ち帰れる
- **勝利**: 最終階層をクリア

## アイテム

- 敵を倒すと確率でアイテムをドロップ
- アイテム種別: ポーション（HP回復）、武器（攻撃力上昇）、鍵（特殊用途）
- 詳細は今後策定（→ 別途Issue化を検討）

## 発話システム

キャラクターがさまざまな場面でセリフを発する。プレイヤーキャラクター・拠点NPCに加え、ユーザー（プレイヤー自身）への語りかけも行う。

### 表示方法

場面に応じて2種類の表示を使い分ける。

| 場面 | 表示方法 | 説明 |
|------|---------|------|
| ダンジョン内 | 頭上吹き出し | キャラクターの頭上に短い一言を表示 |
| 拠点・メタ的発話 | HUD下部テキスト | 画面下部にやや長めのセリフを表示 |

### 発話タイミング

#### ダンジョン内（頭上吹き出し）

- フロア開始時
- 敵発見時
- 戦闘開始時
- 戦闘勝利時
- ダメージを受けた時
- HP低下時（危険域）
- アイテム発見・取得時
- 罠発動時
- 階段発見時
- 帰還判断時（上り階段に接近）

#### 拠点（HUD下部テキスト）

- 帰還時
- ゲームオーバーからの帰還時
- NPC会話
- アップグレード購入時
- ダンジョン出発時

#### メタ的発話（HUD下部テキスト）

- ゲーム起動時
- 長時間操作なし時
- 実績的イベント時（初クリア、初帰還など）

### キャラクター性格

- プレイヤーキャラクターには性格が割り当てられ、同じ発話タイミングでも口調が変化する
- 性格の種類は別途策定（→ 別途Issue化を検討）
