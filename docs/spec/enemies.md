# 敵種別

## 敵タイプ一覧

### 通常敵（スライム）

- **索敵範囲**: 同じ部屋内のみ
- **部屋またぎ**: なし
- **移動距離**: 1マス/ターン
- **行動パターン**: 同じ部屋内にプレイヤーがいれば接近、隣接時に攻撃
- **役割**: 基準バランスの敵。序盤から出現

### 重装敵（ゴーレム）

- **索敵範囲**: マンハッタン距離2マス以内
- **部屋またぎ**: なし
- **移動距離**: 1マス/ターン
- **行動パターン**: 索敵範囲内にプレイヤーがいれば接近、隣接時に攻撃
- **役割**: 高耐久・高火力。間合い管理を促す

### 俊敏敵（コウモリ）

- **索敵範囲**: マンハッタン距離7マス以内
- **部屋またぎ**: あり（通路を通じて追跡）
- **移動距離**: 2マス/ターン
- **行動パターン**: 索敵範囲内にプレイヤーがいれば接近、隣接時に攻撃
- **役割**: 低耐久・高機動。素早い判断を要求

## 階層別配置ルール

| 階層 | 出現する敵タイプ |
|------|-----------------|
| 1〜3F | 通常敵 |
| 4〜6F | 通常敵、重装敵 |
| 7F〜 | 通常敵、重装敵、俊敏敵 |

## 敵AI

1. 索敵条件を判定（敵タイプごとに異なる）
2. 索敵条件を満たす場合: 隣接していれば攻撃、そうでなければプレイヤーに向かって移動
3. 索敵条件を満たさない場合: 待機（移動しない）

経路探索にはBFS（幅優先探索）を使用。
